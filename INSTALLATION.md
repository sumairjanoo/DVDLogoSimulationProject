# Installation Guide

## Requirements
- Python 3.7+
- Pygame library

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/dvd-logo-simulation.git
   ```
2. Install Pygame:
   ```bash
   pip install pygame
   ```
3. Run the simulation:
   ```bash
   python dvd_simulation.py
   ```
